# Sales Performance Dashboard
An end-to-end data analysis project on retail sales data.
## Problem Statement
What drives revenue and profit? Where are losses occurring?
## Dataset
Superstore Sales (Kaggle) — 9,994 rows, 21 columns
## Key Findings
- Technology drives 37% of revenue but Furniture has negative profit in East
- High discounts (>30%) correlate with -15% profit margin
- Q4 is consistently peak season across all years
## Tech Stack
Python · Pandas · Plotly Dash · Jupyter
